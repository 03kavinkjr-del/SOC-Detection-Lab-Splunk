\# Incident Response Playbook



\## 1. Alert Triggered



Review:



\- Alert Name

\- Time

\- Host

\- User



\---



\## 2. Investigate



Check:



\- Parent Process

\- Command Line

\- Process Tree

\- Related Events



\---



\## 3. Validate



Determine whether activity is:



\- Expected

\- Suspicious

\- Malicious



\---



\## 4. Contain



If malicious:



\- Isolate endpoint

\- Disable compromised account

\- Preserve logs



\---



\## 5. Recover



\- Remove malicious artifacts

\- Patch systems

\- Reset credentials



\---



\## 6. Lessons Learned



Document:



\- Root Cause

\- Indicators of Compromise

\- Detection Improvements
