\# Project Goals



The purpose of this project is to build an enterprise-style Security Operations Center (SOC) lab.



Goals:



\- Deploy Splunk Enterprise

\- Collect Windows Event Logs

\- Deploy Sysmon

\- Forward endpoint telemetry

\- Build detection rules

\- Create SOC dashboards

\- Investigate security events

\- Map detections to MITRE ATT\&CK
